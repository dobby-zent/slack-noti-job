# Slack Noti Job

## Dependency

- github.com/ashwanthkumar/slack-go-webhook
- go get -u github.com/spf13/cobra@latest
- go get github.com/spf13/viper

## 실행방법

```sh
    ## for linux / window
    make build

    ## for mac
    make build-arm

    ## move to local
    mv slack /usr/local/bin

    slack --help
```
